MFC 디지털 시계 만들어보기

화면에 디지털시계를 출력하는 프로그램을 작성하는 것. 

1. 프로그램을 실행시키면 "년, 월, 일, 시, 분, 초" 형태로 1초 간격으로 현재 시간을 출력.
2. 왼쪽 마우스 버튼을 누르면 메시지 박스를 이용해 시계의 표시 형태를 "년, 월, 일, 시, 분, 초" 형태와 "시, 분, 초" 형태로 변경.
3. 오른쪽 마우스 버튼을 누르면 시계의 동작 여부를 변경.
4. "시, 분, 초" 형태로 표시할 경우 시간은 24시간이 아닌 12시간으로 표시하고, 오전(AM), 오후(PM)을 구분.
5. 오른쪽 마우스를 누르면 시계가 멈춘 상태면 동작시키겠다는 메시지를 출력하고, 시계가 동작 중이면 시계를 멈추겠다는 메시지를 출력.

목적 : 마우스 메시지와 메시지에 대응하는 메시지 핸들러 함수를 만드는 내용을 익힐 수 있다.
